## 🏗️ Dernier Projet : FacturaPro

> Application SaaS de facturation pensée pour les PME et freelances d'Afrique de l'Ouest.

**Ce que j'ai construit seul, de zéro :**

- 🔐 Authentification email + Google OAuth avec session unique anti-partage
- 📄 Génération PDF (logo, NINEA/NIU, RC, TVA par ligne)
- 💬 Envoi automatique par WhatsApp & Email (4 templates formatés)
- 💰 Suivi des paiements partiels/totaux, verrouillage automatique
- 🌍 Multi-pays (Sénégal 🇸🇳 / Congo 🇨🇬) avec détection automatique
- 📊 Dashboard KPIs, graphiques revenus, alertes retards
- 🔔 Limites d'abonnement, plans TRIAL/PRO, paiement Wave intégré
- ☁️ Déployé sur Vercel avec CI/CD automatique
